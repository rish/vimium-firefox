Keys providen by Vimium-firefox:
  * f - follow link
  * F - follow link in new tab
  * yf - copy link
  * yy - copy current location
  * Ctrl+[ - blur from HTML or XUL control
  * gg - scroll to top
  * G - scroll to bottom
  * h,j,k,l - scroll to left/top/bottom/right

Useful default keys:
  * Alt+Left - Go back
  * Alt+Right - Go forward
  * Ctrl+L - focus on address bar
  * Ctrl+R - reload page
